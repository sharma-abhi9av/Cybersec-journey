# Recon Tracker

A CLI tool for tracking penetration testing targets and findings.
Built for personal use during HTB machines and web app assessments.

## Features
- Add targets (IP / domain)
- Log findings per target
- View full target details
- Delete targets
- Persistent JSON storage — survives restarts

## Usage
python3 recon.py

## Example
Add target: 10.10.10.1
Add finding: port 80 open - Apache 2.4
View target: see all findings listed cleanly

## Built with
Python 3 — zero external dependencies
